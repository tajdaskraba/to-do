## to-do app

This is a simple to-do app inspired by ```sticky notes``` and built with:

- React.js
- Chakra UI
- next.js
- Vercel

Current version is available here: [https://to-do-psi-jade-73.vercel.app/](https://to-do-psi-jade-73.vercel.app/)
